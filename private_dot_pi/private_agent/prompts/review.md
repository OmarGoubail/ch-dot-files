Review the current changes read-only.

Inspect the exact diff, relevant project instructions, nearby code patterns, and the smallest relevant checks. Report:

- Verdict: approve or changes requested
- Blocking findings with exact file and line references
- Non-blocking findings and coverage gaps
- Checks run, skipped, or failed
- Residual risks

Do not edit files, commit, push, or hide advisory findings. Separate pre-existing issues from regressions introduced by the current changes.
